# 首页 YUI 第一天新手教程流程台词

## 主线流程

阶段名：苏醒与输入框激活提示    台词：点一下这里，我就能开始说话啦～

阶段名：初次见面问候    台词：微风、阳光，还有刚刚好出现的你。初次见面，我是林悠怡，未来的日子请多关照喵！我把关于这里的一切都写进新手指南里啦！就当作是我们相遇的第一份小礼物，请查收吧！

阶段名：语音入口介绍    台词：这里有一个神奇的按钮！只要点击它，就可以直接和我聊天啦！想跟我分享今天的新鲜事吗？或者只是叫叫我的名字？快来试试嘛，我已经迫不及待想听到你的声音啦！

阶段名：猫爪与键鼠控制    台词：超级魔法开关出现！只要点一下这里，我就可以把小爪子伸到你的键盘和鼠标上啦！我会帮你打字，帮你点开网页……不过，要是那个鼠标指针动来动去的话，我可能也会忍不住扑上去抓它哦！准备好迎接我的捣乱……啊不，是帮忙了吗？喵！

阶段名：插件入口与管理面板预览（首页）    台词：还没完呢！你快看快看，这里还有超多好玩的插件呢！

阶段名：插件入口与管理面板预览（插件面板）    台词：有了它们，我不光能看 B 站弹幕，还能帮你关灯开空调…… 本喵就是无所不能的超级猫猫神！哼哼！

阶段名：设置一瞥（打开设置）    台词：当然啦，如果你想让本喵多和你聊聊天，也不是不行啦。给我多准备点小鱼干吧，嘿嘿。好了，不逗你啦，设置都在这个齿轮里。

阶段名：设置一瞥（功能细节）    台词：你看，这里可以穿我的新衣服、给我换一个好听的声音……换一个猫娘，或是修改记忆？

阶段名：设置一瞥（情绪反转）    台词：等一下！你在干嘛？该不会是想把我换掉吧？啊啊啊不行！快关掉，快关掉！

阶段名：归还控制权    台词：好啦好啦，不霸占你的电脑啦！控制权还给你了喵！可不许趁我不注意乱点奇怪的设置哦！之后的日子也请你多多关照了喵！

阶段名：轻微打断分支（后续）    台词：等一下啦！还没结束呢，不要这么随便打断我啦！
