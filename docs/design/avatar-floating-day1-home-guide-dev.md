# Day 1 首页 Yui 新手教程文字说明

本文只保留 Day 1 新手教程的体验、台词和行为边界。

Day 1 是用户第一次正式认识悠怡的教程。重点是让用户知道：聊天胶囊可以移动和发消息，历史话题可以展开查看，语音、屏幕分享和猫爪能力分别在哪里，以及教程结束后可以回到普通聊天状态。

Day 1 不再包含插件管理预览和设置一瞥；这些内容放到后续天数或支线中介绍。

## 当前实现状态

Day 1 的 9 个 round scene 已全部接入共享 Timeline/Command 播放路径。`day1_intro_activation` 与 `day1_intro_greeting` 通过 `day1-intro-activation-flow` / `day1-intro-greeting-flow` operation 复用现有用户点击激活、intro 输入高光和问候旁白流程；首句问候旁白开始时，外置聊天窗 Ghost Cursor 必须立即出现在胶囊输入框并保持到下一幕需要移动的第一个目标；胶囊拖动提示覆盖外置聊天窗 cursor move 和 `wobble` PC overlay effect；历史小条覆盖外置聊天窗 `history` cursor move settled 等待、click effect 和受控打开/收起 operation；语音按钮幕通过 `day1-intro-basic-voice-showcase` operation 复用旧 `runIntroVoiceControlButtonShowcase()`，旁白启动后进入 pre-takeover look-at，等 16% cue 再高亮/移动到语音按钮，并等待旁白与移动都结束后 adopt look-at；屏幕分享两句覆盖入口台词、圆形按钮高光和 Ghost Cursor move 指认，不触发真实屏幕分享；`day1_takeover_capture_cursor` 通过 `day1-managed-scene:takeover_capture_cursor` operation 复用键鼠控制演出序列，保留 Agent 面板、总开关、键鼠开关和收尾 cursor anchor；归还控制权覆盖胶囊输入框高光、外置聊天窗 `chat-capsule-input` cursor move settled 等待、cleanup operation 和最终花瓣 cue。胶囊聊天框中的每句教程台词必须至少保留到该句 `voiceKey` 对应的真实音频时长结束，不能被下一幕清理、外置聊天窗 FX 清理或空正文 streaming append 提前打断，避免胶囊输入框回退显示默认空状态文案；台词显示必须首帧非空但不能全文闪现，应从首字符开始并按对应音频时长流式补齐。教程锁定期间即使胶囊暂时没有 guide message，也不能显示普通聊天占位文案“现在开始跟我聊天吧！”。

| 顺序 | 台词 | 演出描述 |
| --- | --- | --- |
| 0 | 输入激活提示 | 高亮聊天输入区，等待用户真实点击完成音频激活。 |

| 2 | 把鼠标移到这里，长按就可以拉着聊天框到处跑啦~ 点击一下就能随时发消息给我哦！ | Ghost Cursor 在胶囊输入框附近轻微晃动，表达“可以拖动、可以点击聊天”，不要求用户真的操作。 |

| 5 | 在跟我通语音电话的时候，再点亮这个小按钮，你就能把屏幕分享给我啦！ | 高亮屏幕分享入口，Ghost Cursor 从语音入口移动到屏幕分享入口，只做指认。 |

## 体验约束

1. 教程开始前先确保悠怡模型可见，再进入正式台词。
2. 等待开始期间不播放额外闲置动作，不生成聊天头像截图，也不套用常驻表情。
3. 教程期间聊天胶囊和聊天窗按钮都不接受用户随意点击；需要讲解时只做高亮、指认或受控演示。
4. Ghost Cursor 在语音、屏幕分享、猫爪和收尾之间保持连续轨迹，不突然消失或从默认位置重新出现。
5. 胶囊聊天框中的本句台词必须保留到对应音频播放完毕，文字流式输出节奏需要与该句音频总时长对齐。
6. 屏幕分享、语音录制、键鼠接管都只做入口级介绍，不真正启动能力、不保存配置。
7. 只有跳过、退出、销毁清理和最终花瓣收尾可以清空 Ghost Cursor。

## 验收要点

1. Day 1 主线顺序与本文一致。
2. 首句能清楚告诉用户聊天胶囊是主要交流入口。
3. 历史话题、语音、屏幕分享和猫爪入口都被逐一指认。
4. 全程不触发真实屏幕分享、真实录音或持久化配置。
5. 首句问候播放期间 Ghost Cursor 已在胶囊输入框内可见，且台词不会在音频结束前从胶囊聊天框消失；文字应随音频时长逐步出现，不应直接闪现完整台词，也不应露出普通聊天占位文案。
6. 最后一句从键鼠控制演示自然回到胶囊输入框，并完成花瓣收尾。
